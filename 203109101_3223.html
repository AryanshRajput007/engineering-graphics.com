  10. Explain Aligned Dimensioning system.

    11. ANS. Aligned dimensioning system is a method of dimensioning where dimensions are placed parallel to the dimension lines, avoiding overlapping and providing clear and readable dimensioning information for objects in engineering and technical drawings.

  11. Explain Unidirectional Dimensioning system.

    12. ANS. Unidirectional dimensioning is a method of dimensioning where all dimensions are aligned in a single direction, either horizontal or vertical. It provides clarity and readability in engineering drawings by eliminating dimension line intersections and ensuring consistent orientation.

  12. What is the position with respect to reference plane , If the object lies in the fourth quadrant ?

    13. ANS. If an object lies in the fourth quadrant, it is positioned below the reference plane.

  13. Explain the term H.C.P.

    14. ANS. In engineering drawings, the term H.C.P. typically refers to the abbreviation for "Horizontal Centerline Plane." It is a reference line or plane used to establish the horizontal centerline of an object or component. The H.C.P. is often indicated in technical drawings to provide a baseline for dimensioning, alignment, and positioning of various features and elements. It helps ensure accurate and consistent measurements and relationships within the drawing.

The representative factor, also known as the scale factor or scale ratio, is a numerical value that represents the relationship between the dimensions of an object in a drawing and its actual size in real life.
